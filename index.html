Firebase 설정 방법

  1. console.firebase.google.com 접속
  2. 프로젝트 만들기 클릭 → 프로젝트 이름 입력 (예: ourboard)
  3. Google Analytics는 사용 안해도 됨 → 프로젝트 만들기
  4. 왼쪽 메뉴 "빌드" → Firestore Database 클릭
  5. 데이터베이스 만들기 → 테스트 모드 선택 → 위치 선택 → 완료
  6. 왼쪽 메뉴 "프로젝트 설정" (톱니바퀴 아이콘)
  7. 하단 "내 앱" → </> (웹) 아이콘 클릭
  8. 앱 닉네임 입력 → 앱 등록
  9. firebaseConfig 객체 내용 복사
  10. 이전 화면으로 돌아와 붙여넣기
